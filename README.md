# keyforge

Keyforge is a growing suite of security tools designed to help users create, validate, and protect sensitive credentials. The current version features an interactive password validator that enforces strong password rules and allows users to export a redacted password hint for safe reference. Future plans include expanding the toolkit with encryption and hashing utilities, all built around privacy-preserving methods. The goal of Keyforge is to make security best practices accessible through simple, well-designed tools.

## Features

- **Password Validator** — real-time validation for length, uppercase, and numeric requirements
- **Hint Export** — export a partially redacted password hint as a `.txt` file

## Getting Started

```bash
pip install flask
python app.py
```

Then open `http://localhost:5000` in your browser.
